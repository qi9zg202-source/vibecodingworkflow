LangGraph UI / Service Links

当前项目本地 `langgraph dev` 已启动。下面表格汇总了可直接访问的 LangGraph 页面与服务地址， 使用 IBM/Carbon 风格表格展示，方便你快速打开 Studio、API Docs 和本地服务。

Start LangGraph Dev Server

如需重新启动当前项目的 LangGraph 本地开发服务，直接在终端执行下面命令。 

cd /Users/beckliu/Documents/0agentproject2026/vibecodingworkflow
source .venv/bin/activate
langgraph dev
Open start-langgraph-dev.command Open scripts folder

说明：浏览器页面不能直接执行本机命令，所以这里提供本地启动脚本。 双击或打开 `start-langgraph-dev.command` 后，会拉起 Ghostty 并执行当前项目的 `langgraph dev`。
