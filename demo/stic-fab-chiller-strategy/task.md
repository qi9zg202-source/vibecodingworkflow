# task.md

## Title
STIC 北方创新中心 Fab 厂务机械课高效制冷机房策略优化建议管理功能

## Goal
建设一个面向厂务机械课的高效制冷机房策略优化建议管理功能，覆盖：
1. 高效制冷机房核心指标曲线监控分析
2. 优化策略包管理
3. 策略包下发、反馈与稳态验证闭环

## In Scope
- EER / COP / 负荷 / 电量 / 预警等核心指标展示与趋势分析
- 策略包列表、详情、收益区间、风险等级、适用工况管理
- 策略下发、执行反馈、稳态验证、回退与闭环状态跟踪

## Out Of Scope
- 直接接管底层控制系统
- 自动执行不可回退控制动作
- 覆盖全厂所有 CUS / HVAC / 公辅子系统

## Constraints
- 必须保留人工审核、人工确认和回退入口
- 所有策略建议必须带假设条件和适用边界
- 必须考虑 N+1、供回水温边界、最小流量边界、稳定性风险
- 必须支持业务闭环追踪，而不只是静态建议展示

## Acceptance Criteria
- 能查看高效制冷机房核心指标趋势和异常预警
- 能查看和管理策略包，并看到适用工况、ROI 范围、风险提示
- 能完成“下发 -> 反馈 -> 稳态验证 -> 完成 / 回退”闭环
- 能保留策略执行记录、反馈记录和稳态结论
