# task.md

## Title
TSMC Fab CUS 制冷站节能策略编排与执行验证工作台

## Goal
建设一个面向厂务机械课的策略工作台，用于管理 Fab CUS 制冷站的节能优化建议，覆盖：
1. 制冷站核心运行状态与约束边界监控
2. 策略包选择、收益测算与风险审查
3. 策略审批、下发、稳态验证、回退与审计闭环

## In Scope
- 中温环路、低温环路、热回收协同相关的策略建议管理
- 负荷、供回水温、流量、机组台数、EER/COP、湿球条件、电价时段等关键输入展示
- 策略包列表、详情、适用工况、节能区间、风险等级、审批与执行记录
- 从“待评估 -> 待审批 -> 待下发 -> 执行中 -> 稳态验证 -> 完成 / 回退”的业务闭环
- 季节性业务场景验证：冬季自然冷却优先、过渡季混合模式、夏季高负荷 N+1 约束

## Out Of Scope
- 直接写入 DCS / PLC 底层控制指令
- 实时自动优化闭环控制
- 覆盖全厂所有 HVAC / 公辅系统
- 替代真实 historian、BMS、工单或审批平台

## Constraints
- 必须保留人工审批、人工回退和操作审计入口
- 必须显式展示负荷边界、供回水温边界、最小流量、N+1 冗余和工艺安全约束
- 策略收益必须以范围估算表达，并注明假设条件（运行时长、电价、环境条件）
- 必须支持稳态观察窗口和执行后结论记录，而不只是静态策略展示

## Acceptance Criteria
- 能查看制冷站关键指标、边界约束和异常提示
- 能基于季节 / 负荷 / 湿球条件查看策略包及其 ROI、风险与回退说明
- 能完整演示“策略建议 -> 审批 -> 下发 -> 反馈 -> 稳态验证 -> 完成 / 回退”链路
- 能保留执行记录、审批记录、稳态结论和审计日志
- 能覆盖至少 3 个真实业务验证场景和 3 类高风险边界样例
